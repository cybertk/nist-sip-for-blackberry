Contains RI, TCK, examples, tools for JAIN-SIP-1.2 (JSR-32 maintenance release) and an SDP library that conforms to the public release of JSR 141 (JAIN-SDP) interfaces. JAIN-SIP RI is a full implementation of RFC 3261. JAIN-SDP interfaces are still under public review and are subject to change. Tools include a signaling trace viewer that can take input from an etherreal/stack trace. RI, TCK and tools are in the public domain.

News

  • The TCK for JSIP-1.2 is Complete! If you are implementing this specification, you can test your implementation against the TCK
  • After much tweaking and optimizing the JAIN-SIP 1.2 RI performance has improved considerably. It measures 75 CAPS on a single processor 2 Ghz Pentium 5 with 1 Gb of RAM. Image size reaches about 130 Megabytes. The stack measured over 200 CAPS on a 4 processor Netra. No memory leaks were noted over 1000000 calls.
  • The JAIN-SIP wiki is making progress. Contribute material an how to tips to the JAIN-SIP Wiki !

Related projects

A usable subset of this project has been "ported" to j2me. Current work in progress includes adding interfaces to support JSR 180 (SIP For J2ME). However, the stack (without the JSR 180 interfaces) is use-able as is. Join the
sip-for-me project on java.net as an observer to get the code from cvs on java.net.

You can imbed the JAIN-SIP stack in an applet, HTTP servlet or an application - check out
jain-sip.applet.phone.dev.java.net
See JAIN-SIP-PRESENCE-PROXY for a proxy server and IM client using JAIN-SIP.
See the SIP Communicator Project on java.net for a soft phone built on JAIN-SIP.
Check out some other cool projects that use JAIN-SIP. http://www.cafesip.org/projects.html
